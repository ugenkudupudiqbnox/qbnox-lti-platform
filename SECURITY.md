# Security Policy

## Supported Versions
The latest released version is actively supported with security fixes.

## Reporting a Vulnerability
Please **do not open public issues** for security concerns.

Instead, report vulnerabilities privately via:
- Email: ugen@qbnox.com

We aim to acknowledge reports within **72 hours**.

## Responsible Disclosure
We follow responsible disclosure practices and will coordinate
patch releases before public disclosure.

Thank you for helping keep the ecosystem secure.
