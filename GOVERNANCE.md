# Project Governance

## Overview
This project is an open-source implementation of an LTI 1.3 + Advantage
platform for Pressbooks. It follows a **benevolent maintainer** model.

## Roles
### Maintainers
- Review and merge pull requests
- Manage releases and security fixes
- Maintain architectural integrity

### Contributors
- Submit issues and pull requests
- Propose enhancements and fixes

## Decision Making
- Minor changes: maintainer approval
- Major changes: design discussion + maintainer consensus
- Security issues: private disclosure to maintainers

## Code of Conduct
This project follows the Contributor Covenant.
